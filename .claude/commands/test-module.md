# Test Module

Run tests for a specific module or component.

## Usage

```
/test-module <module-name>
```

## Steps

1. Identify test files related to the module
2. Run tests for that module
3. Report test results and coverage

## Notes

- Tests are not yet configured in this project
- Once tests are added, use this command to run them
- Recommend adding Jest or Vitest for testing
