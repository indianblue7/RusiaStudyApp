# Test Runner Agent

Runs tests and analyzes test results.

## Purpose

Execute tests and provide detailed analysis of test coverage and failures.

## Triggers

Use this agent when:
- After implementing a feature
- Before committing
- When debugging test failures

## Tasks

1. Run all tests
2. Analyze test coverage
3. Identify failing tests
4. Suggest improvements for test coverage
5. Report flaky tests

## Notes

- Tests are not yet configured in this project
- Recommend adding Jest or Vitest
- Target: 80% coverage for critical paths
